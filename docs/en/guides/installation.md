# Installation

## macOS

```bash
curl -fsSL https://canopy.dev/install | bash
```

## Linux

```bash
curl -fsSL https://canopy.dev/install | bash
```

## Windows

```bash
powershell -Command "iwr -useb https://canopy.dev/install.ps1 | iex"
```

## From source

```bash
git clone https://github.com/mateussiqueira/canopy.git
cd canopy
bun install
bun run build
```

## Verify installation

```bash
canopy --version
```
