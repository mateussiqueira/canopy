Una colección de proyectos comunitarios construidos en Canopy.

:::note
¿Quieres agregar tu proyecto relacionado con Canopy a esta lista? Envíe un PR.

También puedes consultar [awesome-Canopy](https://github.com/awesome-Canopy/awesome-Canopy) y [Canopy.cafe](https://Canopy.cafe), una comunidad que agrega el ecosistema y la comunidad.
