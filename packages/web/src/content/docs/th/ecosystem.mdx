ชุดโครงการชุมชนที่สร้างขึ้นบน Canopy

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ Canopy ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-Canopy](https://github.com/awesome-Canopy/awesome-Canopy) และ [Canopy.cafe](https://Canopy.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
