# Job Search Preparation

Use this skill when the user needs help with job search, career transition, resume/CV optimization, LinkedIn profile, cover letters, interview preparation, or follow-up management.

## Core positioning

Canopy's job search preparation is a **complete process**, not a resume tool:
1. **Material** — Resume, LinkedIn, portfolio, references
2. **Discovery** — Find matching opportunities
3. **Application** — Tailor and submit
4. **Tracking** — Follow-up pipeline
5. **Interview** — Preparation and debrief
6. **Next actions** — Continuous improvement

## Resume (CV) optimization

### Structure
- **Header**: Name, title, location, phone, email, LinkedIn, portfolio
- **Professional summary**: 3-4 lines, keyword-rich, role-targeted
- **Core competencies**: 8-12 skills in 2 columns
- **Experience**: Reverse chronological. Each role: company, location, dates, 4-6 bullet points with metrics
- **Education**: Degree, institution, year. GPA if > 3.5
- **Certifications**: Relevant only
- **Languages**: Proficiency level

### Bullet points formula
```
{Action verb} {task} {result/metric} {tool/methodology}
```
Examples:
- "Reduced processing time by 40% by implementing automated ETL pipelines with Python and Airflow"
- "Led cross-functional team of 5 engineers to deliver $2M project ahead of schedule"

### ATS optimization
- Use standard section headings ("Experience", not "Work History")
- Include keywords from the job description verbatim
- Avoid tables, columns, graphics, headers/footers
- Save as .docx for ATS systems that reject PDFs
- Use `.docx` extension, NOT `.doc`

## LinkedIn optimization

### Headline
- Format: `[Role] | [Specialty] | [Key metric/achievement]`
- Max 220 characters
- Include target keywords

### About section
- First 2 lines visible without "see more" — hook here
- Structure: Problem → Solution → Proof → CTA
- 3-5 paragraphs, 2000 chars max

### Featured
- Add top 3 achievements: articles, projects, recommendations
- Use rich media (PDF, video, links)

### Experience
- Same bullet formula as resume
- Add media to each role (presentations, documents)
- Get 2-3 recommendations from managers/peers

### Skills & endorsements
- List 50 skills relevant to target role
- Reorder: most important top 3
- Ask for endorsements from key connections

## Cover letter

### Structure
1. **Hook**: Specific achievement or connection to the company
2. **Why you**: 2-3 relevant accomplishments with metrics
3. **Why them**: Demonstrate company research
4. **Close**: Call to action, confidence statement

### Rules
- Max 400 words
- Address to specific person when possible
- Never generic — reference job description
- PDF format for submission

## Interview preparation

### Before
- Research company (recent news, culture, competitors)
- Review job description — prepare 3 stories per requirement
- Prepare STAR stories: Situation, Task, Action, Result
- Prepare 3 questions for the interviewer

### Common questions
- "Tell me about yourself" → 60-second career narrative
- "Why this company?" → Specific, researched, genuine
- "Biggest achievement" → STAR story with metric
- "Weakness" → Real weakness + action taken + improvement

### Follow-up
- Thank-you email within 24h
- Reference specific conversation points
- Reiterate enthusiasm and fit

## Pipeline tracking

Track each opportunity through stages:
1. Applied
2. Screening
3. Interview (1st, 2nd, Final)
4. Offer
5. Accepted/Declined

For each: company, role, contact, date of last action, next action, notes.

## Templates

### Follow-up email (after interview)
```
Subject: Thank you — [Role] interview

Hi [Name],

Thank you for the opportunity to discuss the [Role] position. I especially enjoyed learning about [specific topic from interview].

I'm very excited about [specific aspect of role/company] and believe my experience with [relevant skill] would make me a strong addition to the team.

Please let me know if you need any additional information.

Best,
[Your name]
```

### Connection request (LinkedIn)
```
Hi [Name],

I'm impressed by your work in [specific area]. As a [your role] specializing in [your specialty], I'd love to connect and follow your insights.

Best,
[Your name]
```
